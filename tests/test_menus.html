DropDowns

Toggles overlays for displaying lists of items and other elements.

Overview

The dropdown component creates buttons that toggle contextual overlays. These overlays can be used to display a list of links or other elements. They are designed to be as customizable as possible. They can be configured to toggle on on user click on on mouse over and many other configuration properties are available.

Dropdown can be used to display any type of overlay element. The most common display of the dropdown component is the Menu component but any element can be used as long as it defines the attribute data-role="menu". When the dropdown toggles it first searches for it submenu child. The submenu child is defined as any element that has the data-role="menu" attribute. Once found it checks to see if it has an attached menu component. If it does it calls it's show method. Otherwise, it toggles the visible and hidden utility classes as needed.

Timeout

If you want your dropdown menu to automatically deactivate after a set period of time after the user leaves the item, you can set it's timeout property. The timeout property takes an integer greater then or equal to 0. After the user leaves the item the dropdown is deactivated after that many milliseconds. If the user hover back over the dropdown before it expires the timer is cleared.

Properties

Property Name Attribute Name Description
closeOnBlur data-close-on-blur Boolean
When true the dropdown or menu closes when the user clicks outside of it.
closeOnSelect data-close-on-select Boolean
When true the dropdown closes when the user selects an item.
timeout data-timeout false|integer
Defines the number of milliseconds after the user leaves the item that it deactivates. If false or a negative number then the menu will never deactivate.
autoActivate data-auto-activate false|integer
Defines the number of milliseconds after the user enters the item that it activates. If false or a negative number the dropdown will never activate. To tell the dropdown to activate immediately set the value to 0.
toggle data-toggle on|off|both
Controls how the dropdown handles being clicked. If on then the dropdown will toggle on when clicked but won't toggle off when clicked. If off the reverse happens. If both then the dropdown will toggle both on and off.
