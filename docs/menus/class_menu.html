Menu

A component for rendering nestable list of selectable items.

Properties

boolean toggleOn True if menu items should toggle on.
@readonly
boolean toggleOff True if menu items should toggle off.
@readonly
MenuItem[] activeItems A list of all active menu items.
@readonly
String menuNodeType Set to menu for Menu nodes and it's subclasses.
@readonly
MenuItem MenuItemClass The class that is used to construct new menu items.
Menu MenuClass The class that is used to construct to sub menus.
Boolean closeOnBlur If true the menu will close when the user clicks outside of it.
Boolean|Number timeout

Controls how long after the user leaves the menu that is timeouts.

Values greater then or equal to 0 will cause the menu to timeout after that amount of time has elapsed in milliseconds. Negative values or false will cause the menu to never timeout.
Boolean|Number autoActivate

Controls if child items will activate when the user hovers over them. Can be true, false or a number greater then or equal to 0. Numbers specify that amount of time the user must remain over the item before it activates.

This property isn't used if the menu is active. See openOnHover to configure how items behave when the menu is active.
Boolean|Number openOnHover

Controls if child items will activate when the user hovers over them when the menus state is active.
Value can be either true, false or a number. Numbers less than 0 are equivalent to false.
If true items will activate immediately. If false items will never activate.
If a number greater then or equal to 0 the item will activate after that amount of time in milliseconds passes.
String toggle

Controls if items will activate or deactivate when clicked. The value can be either on, off, both or none.

If on child item will activate when clicked, but will not deactivate when clicked. Off is the opposite and item will not activate when clicked but will toggle off when clicked. Both will cause items to toggle on and off. Finally, none will prevent the items from toggling on or off when clicked.
boolean closeOnSelect If true the menu will deactivate when a select event is encountered.
Number delay Adds a delay between item activation and submenu display.
Function position A function to call when displaying submenus. Controls how that submenu should be positioned.
boolean isVisible True if the menu is visible. False otherwise.
boolean multiple If true mutliple child items will be allowed to be activate at the same time.

Parameters

Object options Configuration object for Menu
DOMElement options.target The target element to use as the widgets dom element.
Boolean options.closeOnBlur If true the menu will close when the user clicks outside of it.
Boolean|Number options.timeout

Controls the amount of time after the user leaves the menu that it will deactivate.

The value can be either be false for never timeout or a number. If a number the menu will timeout after that amount of time in milliseconds has passed.
Boolean|Number options.autoActivate

Controls how child item behave when the mouse hover over them when the menu's state is unactive.

If the menu is active this behavior is controlled by openOnHover instead.

Value can be either a boolean or a number greater then or equal to 0. If a number the menu-item will activate after the given amount of time in milliseconds has passed, if true the item will activate immediately and if false the item will never activate on mouse over.
boolean options.multiple If true multiple child items are allowed to be activated simutaniously.
Boolean|Number options.openOnHover

Controls how child item behave when the mouse hover over them when the menu's state is active. When unactive this behavior is controlled by autoActivate.

Value can be either a boolean or a number greater then or equal to 0. If a number the menu-item will activate after the given amount of time in milliseconds has passed, if true the item will activate immediately and if false the item will never activate on mouse over.
on|off|none|both options.toggle Controls how child item behave when clicked. Can either be on, off, none or both. If on item will only toggle on; If off items will only toggle off when clicked; If none items will never toggle on or off; and if both items will toggle both on and off.
Boolean options.closeOnSelect If true the menu will deactivate when a select event is encountered.
Number options.delay Adds a delay between the activation of menuitems and the displaying of thieir submenus.
String options.id Sets the id of the menu.
String options.classes A list of space seperated classes that will be added to the menus classList.
Object[] options.children An array of child item configuration objects that will be used to construct the menu's children.
boolean options.visible If true the menu will start off as visible.
Function|inherit options.position A optional function that will be called when the menu is displayed. Used to position the menu.
Object options.context Any addtional keyword arguments are captured by the context argument and passed to the render function.
