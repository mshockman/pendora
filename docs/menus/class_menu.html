Menu Class

A component for managing a nestable list of items that map user event's to actions.

autoActivate : {bool|Number}

When the menu is in the unactive state, this property controls if the user can activate the menu by hovering over a menu item. If true the item will immediately activate when the user hovers over it. If an integer >= 0, the menu will activate if the user hovers over the item and stays over the item over that amount of time in milliseconds.

This behavior can be controlled separately for both the active and unactive state. When the menu is active see openOnHover to control this behavior.

openOnHover : {bool|Number}

When the menu is in the active state, this property controls if the user can activate the menu by hovering over a menu item. If true the item will immediately activate when the user hovers over it. If an integer >= 0, the menu will activate if the user hovers over the item and stays over the item over that amount of time in milliseconds.

This behavior can be controlled separately for both the active and unactive state. When the menu is unactive see autoActivate to control this behavior.

closeOnBlur : bool

If set to true, the menu will close when it loses focus. For example, when the user clicks outside it.

Default:
false

timeout : {bool|Number}

Controls the amount of time after the user leaves the menu that it times out.

If a number greater then or equal to 0, the menu will timeout after that amount of milliseconds.

If false or a negative number, the menu will never timeout.

Default:
false

multiple : bool

If true multiple children will be allowed to be active at the same time.

If false whenever a child activates any sibling nodes that are active will be deactivated.

Default:
false

toggle : String

Controls when menu items will toggle active and unactive.

Can be one of three 4 choices. (on, off, both, none) If on the menu will activate when clicked but not deactivate when clicked. When off the menu will do the reverse and deactivate when clicked and but activate. The value both will allow the menu to toggle on and off. And none will prevent the menu from toggling on or off when clicked.

Default:
"on"

closeOnSelect : bool

If true, the menu will close when an item is selected.

Default:
false

deactivateOnItemHover : bool

If true, items will deactivate when the user mouses over a sibling reguardless if it activates or not.

This can be used so that items will still deactivate even if the sibling item you mouse over is disabled.

Only applies if multiple is false.

Default:
true
