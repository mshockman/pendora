Menu Class

A component for managing a nestable list of items that map user event's to actions.

Menus

A component that represents a nestable list of selectable menu items. Menus can be positioned relative to their parent dropdown or menuitem.

Properties

closeOnBlurIf true the menu will close when the user clicks outside of it.
timeoutControls the amount of time after the user leaves the menu that it will deactivate.
autoActivateControls if menu items will activate when the user mouses hover them before the menu has been activate. After activation this is controlled by the openOnHover property.
multipleControls if multiple menu items are allowed to be active at the same time.
openOnHoverControls is menuitems will activate on hover after the menu is active. Before activation this is controlled by autoActivate.
toggleControls how menu items behave when clicked. Allows them to toggle on, off, both or neither.
closeOnSelectIf true the menu will deactivate if it encounters a menuitem.selected event.
deactivateOnItemHover
delayAdds a delay to menu items with dropdown menus between when they activate and when show their submenus.
positionA function that is used to position the menu.
activeItemsA readonly property that contains list of all active child items.
toggleOnA readonly property that is true if menu items should toggle on.
toggleOffA readonly property that is true if menu items should toggle off.

constructor(Object config);

Config Parameters

targetString|HTMLElementThe element to use as the menu.
closeOnBlurBoolean|NumberControls how the menu responds to the user clicking outside it.
timeoutBoolean|NumberControls the amount of time after the user leaves the menu that it deactivates.
autoActivateBoolean|NumberControls how child menu items respond to on mouse over events when the menu is unactive.
multipleBoolean|StringIf true multiple children are allowed to be active simultaneously.
openOnHoverBoolean|NumberControls how children behave on mouse over when the menu is active.
toggleStringControls how children behave when clicked.
closeOnSelectBooleanIf true the menu will deactivate when it received a select event from a descendant.
deactivateOnItemHoverBoolean
delayBoolean|NumberAdds a delay between child item activation and the displaying of sub menus.
idStringAds an id to the menu.
classesStringSpace separated classes to be added to the menu.
childrenArrayAn array of menu item configuration objects to be added to the menu.
visibleBooleanControls if the menu starts off as visible.
positionFunction(menu)A function that will be called to dynamically position the menu.
...contextAny unknown arguments are gathered in a object and passed to render.
