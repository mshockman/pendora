MenuBar

MenuBar is a horizontal list of dropdown menus that can contain navigation links and more. By default, the MenuBar is activated when the user clicks on a root level menu-item. After the MenuBar is activated child items can then be activated on hover. This is designed to simulate the normal behavior of MenuBars found in Window's and Mac applications.
