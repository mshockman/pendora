Menu Overview

A library for creating nested menus & ribbons.

Menus

A flexible menu component that can be used to create common navigation and control patterns.

Menu and it's subclasses can be initialized in one of three different ways. Imperatively using code; from JSON object; or parsed from the DOM. Building your menu imperatively will provide you with the most amount of control. The menu module comes with several intelligent sub classes which construct the most common uses cases. See the table below.

Classes

MenuNodeThe base class for both Menu and MenuItem. Provides the utilities needed to handle events and manage the menu tree of nodes.
MenuItemA configurable button that displays a menu overlay when active. Represents the menu items in all menu classes.
MenuThe Menu class displays an overlay of a list of menu items that the user can select. Used as a submenu for MenuBar, VerticalMenu and DropDown menu classes and is also inherited by all other types of menus.
MenuBarSubclass of Menu. Creates a windows like horizontal menubar for using at the top and bottom of views.
DropDownA subclass of MenuItem that can act as the root node of a menu tree.
