DropDown

Creates a contextual menu that toggles on and off when the user clicks the target button. The dropdown menu can contain any html markup or menu items that can be used for navigation or can have drop down menus of their own creating a list of nested sub menus.

The most common way to use this class is to initialize it from the dom using the data-init="dropdown" attribute. Further sub components of the dropdown component can be marked using the attributes data-menu, data-button, and data-menuitem.

The data-button attribute is used to mark the root button of the dropdown component. It is the button that the user clicks. Every dropdown is required to have one. A dropdown is also required to have a data-menu child. This is the element that will be initialized as the sub menu of the dropdown. Any additional menuitems inside the dropdown should be marked with the data-menuitem attribute so the class knows to initialize them.
