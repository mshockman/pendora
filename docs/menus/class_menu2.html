Menu Class

A component for managing a nestable list of items that map user event's to actions.

Menu

A component for rendering nestable list of selectable items.

Properties

toggleOnTrue if menu items should toggle on.
toggleOffTrue if menu items should toggle off.
activeItemsA list of all active menu items.
menuNodeTypeSet to menu for Menu nodes and it's subclasses.
MenuItemClassThe class that is used to construct new menu items.
MenuClassThe class that is used to construct to sub menus.
closeOnBlurIf true the menu will close when the user clicks outside of it.
timeoutControls how long after the user leaves the menu that is timeouts. Values greater then or equal to 0 will cause the menu to timeout after that amount of time has elapsed in milliseconds. Negative values or false will cause the menu to never timeout.
autoActivateControls if child items will activate when the user hovers over them. Can be true, false or a number greater then or equal to 0. Numbers specify that amount of time the user must remain over the item before it activates. This property isn't used if the menu is active. See openOnHover to configure how items behave when the menu is active.
openOnHoverControls if child items will activate when the user hovers over them when the menus state is active. Value can be either true, false or a number. Numbers less than 0 are equivalent to false. If true items will activate immediately. If false items will never activate. If a number greater then or equal to 0 the item will activate after that amount of time in milliseconds passes.
toggleControls if items will activate or deactivate when clicked. The value can be either on, off, both or none. If on child item will activate when clicked, but will not deactivate when clicked. Off is the opposite and item will not activate when clicked but will toggle off when clicked. Both will cause items to toggle on and off. Finally, none will prevent the items from toggling on or off when clicked.
closeOnSelectIf true the menu will deactivate when a select event is encountered.
delayAdds a delay between item activation and submenu display.
positionA function to call when displaying submenus. Controls how that submenu should be positioned.
isVisibleTrue if the menu is visible. False otherwise.
multipleIf true mutliple child items will be allowed to be activate at the same time.
