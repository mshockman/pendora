Pendora

A Javascript UI library for creating interactive web applications.

Component List

Menu Library

MenuBar A component that creates a horizontal navigation menu filled with nestable pull down items.
DropDown A component that creates a contextual overlay over a button that can be used to display navigation items, forms or any other component.
Select2 An alternative to the built in select element that allow full control over the style and option filtering.
SideMenu A component that creates a contextual overlay over a button that can be used to display navigation items, forms or any other component.
ContextMenu A contextual overlay that can contain nestable sub menus, forms or other components.
