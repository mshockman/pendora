Summary

The Notification class is used to create fully customizable notifications that can be displayed inside an element, overlaying the page, or targeting an element.

Type
Placement
Target

Element Notifications

Element notifications are notifications that display on an element as a tooltip.

{% highlight javascript linenos %} Notification.notify("Hello World", "success", {placement, "top": target: "#target"}); {% endhighlight %}

Global Notifications

Global notifications are notifications that appear at predefined positions on the page. This positions are top-left, top, top-right, left, right, bottom-left, bottom, and bottom-right.

{% highlight javascript linenos %} Notification.notify("Hello World", "success", {placement, "top"}); {% endhighlight %}

Inside Target Element

Notification can also be place inline with the page by specifying an element to append the notification to as the placement.

{% highlight javascript linenos %} Notification.notify("Hello World", "success", {placement, "#selector"}); {% endhighlight %}
