Menu

A flexible menu component that can be used to create common navigation and control patterns.

Menus can be initialized in three different ways. Imperatively, from JSON data or from the DOM. Building your menu imperatively by constructing the classes manually will provide you with the most amount of control but is unnessary for most situations. The menus module comes with several intelligent sub classes which construct the most common uses cases.
