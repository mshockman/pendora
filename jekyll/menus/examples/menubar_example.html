From Markup

To create a Menubar using markup you need to add the data-init="menubar" attribute to the top level node of the menu. From there it will crawl though the tree and register any items or menus.

A Menu or MenuItem is defined when an element is labeled with the data-role="menu" or data-role="menuitem" attribute.

  • Root Item #1
    • Sub Item #1
    • Sub Item #2
    • Sub Item #3
    • Sub Item #4
    • Sub Item #5
  • Root Item #2
    • Sub Item #1
    • Sub Item #2
    • Sub Item #3
    • Sub Item #4
    • Sub Item #5
{% endhighlight %}

Using JSON

{% highlight javascript linenos %} let menubar = new MenuBar(); menubar.createItems([ { text: "Root #1", children: [ { text: "Sub Item #1" href: "url"}, { text: "Sub Item #2" href: "url"}, { text: "Sub Item #3" href: "url"}, { text: "Sub Item #4" href: "url"}, { text: "Sub Item #5" href: "url"} ] }, { text: "Root #1", children: [ { text: "Sub Item #1" href: "url"}, { text: "Sub Item #2" href: "url"}, { text: "Sub Item #3" href: "url"}, { text: "Sub Item #4" href: "url"}, { text: "Sub Item #5" href: "url"} ] } ]); menubar.appendTo(container); {% endhighlight %}

Using Javascript

{% highlight javascript linenos %} let menubar = new MenuBar(); // Create root 1 branch let root1 = menubar.addItem("Root #1"); root1.addItem("Sub Item #1", "url"); root1.addItem("Sub Item #2", "url"); root1.addItem("Sub Item #3", "url"); root1.addItem("Sub Item #4", "url"); root1.addItem("Sub Item #5", "url"); // Create root 2 branch let root2 = menubar.addItem("Root #1"); root2.addItem("Sub Item #1", "url"); root2.addItem("Sub Item #2", "url"); root2.addItem("Sub Item #3", "url"); root2.addItem("Sub Item #4", "url"); root2.addItem("Sub Item #5", "url"); menubar.appendTo(container); {% endhighlight %}
