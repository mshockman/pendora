Select

A replacement for the default select menu.

Overview

Their are many cases where the default behavior of the <select> element just isn't enough. For example, if you want to perform advanced styling on the menu or if it's behavior when multiple is set to true isn't to your liking. The Select2 widget is a replacement for the default <select> element that fixes some of this issues.
