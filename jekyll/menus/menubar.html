MenuBar

The MenuBar widget is used to display a horizontal nav-bar containing pull down items that display nestable lists of menus or selectable items.

Like all menus MenuBar can be initialized either from the dom, imperatively or by using a json config object.

Positioning

Menu items first position their sub menus below themselves at sub level #1. Then any additional submenus below level 1 will position themselves to the right of their parent menu item.

By default, when a menu items submenu would display itself outside of it's container it will flip it's position to the opposite side of the menu item for that axis. You can change this behavior by applying a position function to the menu through it's position parameter.

Making a MenuBar that Opens Automatically on Mouse-Over

By default a MenuBar will only activate once the user as selected a menu-item from it's root menu. You can change the menu to automatically on mouse-over by setting the autoActivate property to a number greater then or equal to 0. This number is the amount of time in milliseconds that the user must hover over an item to activate it.
