MenuBar

Provides a window like top level pull down menus.

Header: import {MenuBar} from menus/MenuBar
Class: MenuBar
Inherits: Publisher/MenuNode/Menu
Auto Loader: menubar

Detailed Description

A menu bar consists of a hierarchy of pull down menu items. Menus can best nested into submenus that the user can navigate. A menu bar can be defined in one of three ways. The first is by importing all items via a json document using menubar.createItems(data) method; The second is by parsing element in the DOM; The final way is imperatively by building a menu using the classes MenuBar, Menu, MenuItem or any other class that descends from MenuNode.
